# Podcast reunion: S Robinem Pokorným o webdesignu a webové vývojařině v roce 2017

Ano, vážení posluchači, podcast se mění. Je mi potěšením vám oznámit, že jej teď budeme hostovat společně [s Robinem](https://robinpokorny.com/). Můžete se tak těšit na častější epizody a širší záběr.

V první společné epizodě jsme toho probrali fakt hodně: Web 1.0 Retro, iRozhlas.cz a reklamu, React Fiber, licenci Reactu, Webassembly, GraphQL, Web Components, WebP a WebM, Bootstrap 4, CSS Grid a mnoho dalšího.

[Stahujte zde](http://cast.rocks/hosting/10893/Prvni-spolecny-s-Robinem.mp3) nebo poslouchejte v přehrávači nebo přes iTunes.
