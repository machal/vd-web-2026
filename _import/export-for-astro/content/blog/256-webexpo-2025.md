# Pozvánka na WebExpo 2025: Bramus Van Damme, Tomáš Čupr, Jan Řezáb, Tammy Everts, Tim Kadlec a další

Prošel jsem si [program letošního WebExpa](https://webexpo.net/prague2025/program/#day1) a už se začal těšit. Jako každý rok. Přináším vám svůj výběr přednášek, které byste podle mě neměli minout.

Letos mě to táhne nejen k tématům rychlosti webu (to je asi jasné) ale daleko víc se dívám i do byznysu (viz moje [transformace](https://www.vzhurudolu.cz/blog/253-2024)). Vývojařinu, CSS a webařinu jako takovou ale po očku stále sleduji, takže v mých tipech nebudou chybět.

WebExpo je pro mě každoročně _must_. Zde nasaju inspiraci, potkám kolegy a kolegyně a taky se dozvím, co je nového ve webovém světě.

![WebExpo 2025](https://res.cloudinary.com/vzhurudolu-cz/image/upload/v1742532209/vzhurudolu-blog/WebExpo_2025_banner_apfqes.jpg)

Pojďme se teď podívat na to, co mě v programu WebExpo 2025 zaujalo.

## Performance {#perf}

1. [**Tammy Everts: Connecting the dots between site speed, user happiness, and business success**](https://webexpo.net/prague2025/sessions/connecting-the-dots-between-site-speed-user-happiness-business-success/)  
   Talk Tammy ze SpeedCurve cílí na propojení rychlosti webu s reálnou spokojeností uživatelů, což je přesně to, co se snažím klientům v poslední době vysvětlovat.

2. [**Tim Kadlec: One step ahead**](https://webexpo.net/prague2025/sessions/tim-kadlec-web-performance-2025/)  
   Tim Kadlec bude mluvit o nových nástrojích a postupech, které dokážou naše weby zrychlit. Weby se sice zrychlují, ale nároky uživatelů rostou.


![WebExpo 2024](https://res.cloudinary.com/vzhurudolu-cz/image/upload/v1742532210/vzhurudolu-blog/First_Stage__Lucerna_Great_Hall_-_Nathan_W._Pyle2_immvql.jpg)

## Vývoj, UI, CSS {#dev}

1. [**Bramus Van Damme: Supercharge web UX with view transitions!**](https://webexpo.net/prague2025/sessions/supercharge-web-ux-with-view-transitions/)  
   Bramus je kouzelník přes CSS, takže se moc těším na jeho triky pro plynulejší a efektnější přechody, které posunou vnímání webu zase o kus dál.

2. [**Vitaly Friedman: Design patterns for search UX in 2025**](https://webexpo.net/prague2025/sessions/vitaly-friedman-web-design-2025/)  
   K tomuhle už snad nemusím nic dodávat. Vitaly vždycky naservíruje spoustu praktických tipů a patří už neoddělitelně ke značce WebExpa.

3. [**Krasimir Tsonev: React in Patterns (workshop)**](https://webexpo.net/prague2025/sessions/premium-workshop-react-in-patterns/)  
   Krasimir možná u nás není tak známý, ale kdo viděl jeho [loňskou přednášku](https://webexpo.net/blog/krasimir-tsonev-revealing-6-signs-you-suffer-from-senior-developer-syndrome/), ví, že má v malíku React a umí skvěle vysvětlovat, jak psát přehledný kód.

4. [**Darko Bozhinovski: Future-proof your UI: Building with Web Components**](https://webexpo.net/prague2025/sessions/future-proof-your-ui-building-with-web-components/)  
   Web Components se už nějakou dobu chystají být „další velká věc“, ale však víte… zatím se jim to zase tak nedaří. Další přednášku z praxe vítám. 

5. [**Shruti Balasa: Flex, Grid, and beyond: Mastering layouts with Tailwind CSS**](https://webexpo.net/prague2025/sessions/mastering-layouts-with-tailwind-css-flex-grid-and-beyond/)  
   Shruti se věnuje Tailwindu dlouhodobě, takže jsem zvědavý, jak nám ukáže kombinaci jeho utilit se současnými layoutovými možnostmi v CSS. Tailwind a grid, to je kombinace, kterou můžu.

6. [**Rowdy Rabouw: Fugu: Possibilities with web capabilities**](https://webexpo.net/prague2025/sessions/rowdy-rabouw-web-development-2025/)  
   Jsem hodně zvědavý, jak se JavaScript a webové API rozšiřují dál do oblastí, kam dřív patřily spíš nativní aplikace – a Rowdy údajně chystá pěkný přehled toho, co se chystá.

7. [**Karel Navrátil: Dark patterns: Where does motivation end and illegal manipulation begin?**](https://webexpo.net/prague2025/sessions/dark-patterns-where-does-motivation-end-and-illegal-manipulation-begin/)  
   Kódujícího advokáta Karla jsme už několikrát měli na Frontendistech a je to rostoucí talent, na který se vyplatí zaměřit.

## Byznys {#byz}

1. [**Tomáš Čupr: Startup founder story**](https://webexpo.net/prague2025/sessions/tomas-cupr-talk-2025/)  
   Když se někdo podílí na tolika úspěšných startupech, chci ho poslouchat.

2. [**Jan Řezáb: How AI can revolutionise startup growth**](https://webexpo.net/prague2025/sessions/jan-rezab-talk-2025/)  
   Umělá inteligence je momentálně strašně žhavé téma a Jan prý ukáže, jak ji reálně zapojit do škálování byznysu, což mě zajímá.
