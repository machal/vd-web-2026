## Měření a optimalizace CLS {#CLS}

Cumulative Layout Shift (CLS) udává stabilitu vzhledu stránky během vykreslování a její špatné hodnoty mohou pro uživatele znamenat velmi nepříjemný prožitek při vykreslování stránky.

### Obsah

1. Co je CLS a proč je důležité?  
Ukázka z denikn.cz, CLS jako součást důležitých Web Vitals, CLS jako UX metrika.
2. Problém a jeho měření  
Měření pomocí PageSpeed Insights, PageSpeed.cz, důležité rozšíření Web Vitals, nepostradatelná Google Search Console.
3. Co by vás nenapadlo  
Jak přesně se layout shifty počítají, 500 ms hranice pro vyžádané posuny, časté problémy s animacemi.
4. Demo  
Případová studie opravy CLS na webu Deníku N a Slevomatu – měříme, hledáme problémy, ověřujeme hypotézy za pomocí DevTools a dalších nástrojů.
5. Optimalizace  
Držíme plochu s poměrem stran, pozor na asynchronní JS, optimalizace nechtěných posuný vyvolaných fonty, potíže s vkládanou reklamou.
6. Otázky a diskuze  
Diskuze nad weby Revolgy, Autobazar Vysočina a Slevomat. Font Metrics Override Descriptor.

## Měříme a optimalizujeme LCP {#LCP}

Largest Contentful Paint je možná tou nejdůležitější ze sady metrik rychlosti webu Web Vitals, doporučovaných Googlem.

### Ukázka

### Obsah

1. Co je LCP  
Důležitost Largest Contentful Paint. LCP v kontextu dalších metrik. Co může a nemůže být „největší obsahový prvek“ a jak jej nalézt.
2. Měříme LCP  
PageSpeed Insights a zmatek v číslech. Měříme lépe na PageSpeed.cz. Proč je Google Search Console nepostradatelná.
3. Demo  
Ukázková pitva a optimalizace dvou obsahových webů a jednoho e-shopu za použití Chrome DevTools a nástrojů Lighthouse, tracingu v Performance tabu.
4. Otázky  
Jak velké zmenšení je potřeba? Úprava designu kvůli LCP. Rozdíl mezi syntetikou a uživatelským měřením.
