# Pár poznámek k průzkumu mezi vývojáři na Stack Overflow

Některé informace z [The 2018 Developer Survey Results](https://insights.stackoverflow.com/survey/2018/) mi přišly zajímavé. Podělím se.

Stack Overflow dělá tyhle průzkumy každoročně. Asi tenhle web i vy používáte několikrát týdně, takže se nemůžete divit, že dostali na 100 tisíc odpovědí. Mimochodem, z Česka mě to vychází na cca 800 lidí (0,79%) ze Slovenska na cca 240 (0,24 %). Následuje ale rozbor věcí, které se mi zdály zajímavé v celosvětových číslech.

<!-- AdSnippet -->

## Technologie: Vede JavaScript a Node.js

Javascript je šest let za sebou nejčastěji používaný jazyk (70 %). Není těsně následován PHP, jak by se z mojí bubliny mohlo zdát, ale samozřejmě HTML, CSS a SQL. Další programovací jazyk je až Java s 45 %. PHP má 31 % podíl v používání.

Je dobré si uvědomit, že průzkum je prováděný mezi *všemi* vývojáři. Ti weboví tvoří jen část dotazovaných. Ale už delší dobou je jasné, že webová trojka – HTML/CSS/JS – získává popularitu i v dříve newebových oblastech.

Mezi frameworky vedou ty javascriptové - jasně první je Node.js s 50 %. Následuje Angular s 37 %. React je až třetí s 27 %. Na druhou stranu: [Loni](https://insights.stackoverflow.com/survey/2017/) měl Angular 44 %, React jen 20. To další komentář nepotřebuje. 

Pro programátory mezi vámi:

* Tři nejoblíbenější jazyky: Rust, Kotlin a Python.
* Tři nejobávanější: Visual Basic 6, Cobol a CoffeeScript. 
* Python je už druhým rokem jazykem, který se lidé nejvíc chtějí naučit.

## Nástroje: Visual Studio Code poprvé v čele

Visual Studio Code je poprvé nejoblíbenější nástroj mezi webovými vývojáři. Následuje klasické Visual Studio, Notepad++ a Sublime Text. Takový Atom je až osmý.

Pokud bylo Visual Studio dlouho první, vůbec se vítězství VS Code nedivím. Jako odlehčený editor typu Sublime prostě vysál uživatele tradičnímu Visual Studio a samozřejmě také ostatním v řadě.

Dvě věci mě zaujaly k hardwarovým nástrojům a ergonomii:

* Více než polovina lidí používá dva monitory. Šestina dokonce tři a více.
* 50,4 % používá stůl pro stání. (Já taky, ale i tak: Wow!)


## Soft věci

83 % říká, že se učí nové nástroje učí z oficiální dokumentace, stejný počet na Stack Overflow. 50 % z knih a e-booků a stejný počet z online komunit mimo Stack Overflow. 

Přijde mi to zajímavé v kontextu mého dotazníku mezi zdejšími webaři. Ptal jsem se totiž trochu jinak. Nejvíce využívané mě vyšly krátké texty (96 %), videa (78 %), sociální sítě a online diskuze (62,5 %). Nad polovinu se dostaly ještě ebooky (51 %).

Zaujaly mě taky dva výsledky dotazů ke kariéře:

* Co byste chtěli dělat za pět let? 34 % by dále chtěli dělat vývojařinu, ale na jiné nebo specializovanější pozici. 25 % chce založit firmu.
* Co vývojářům přijde důležité při hledání pracovní místa? Skoro stejně důležité jsou prachy a benefity (18,3 %) a jazyky nebo frameworky, které firma používá (17,3 %). 16 % kouká na příležitosti pro profesní rozvoj. Jen 6,5 % např řeší, jak širokou skupinou lidí bude produkt používaný.
