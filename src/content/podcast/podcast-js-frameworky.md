# Podcast: S Riki Fridrichem o současnosti a budoucnosti javascriptových frameworků

Riki je javascriptový vývojář, který nedávno začal [školit Vue.js](https://www.vzhurudolu.cz/kurzy/vue-js). Robin je také javascriptový vývojář. A Martina zajímá stav a budoucnost javascriptových frameworků. Tak jsme to dali dohromady.

Procházíme krajinou frameworků od Angularu.js, přes dnes nejrozšířenější React a rychle rostoucí Vue.js až po novinky v podobě Svelte. Řešíme jejich silné stránky a další důvody, proč si je vývojáři vybírají. A také co který framework ukradl ostatním.

Jedna otázka zůstává: pohádá se Riki s Robinem?
