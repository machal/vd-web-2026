# Podcast: S Martinem Krištofem o frameworku Next.js

O frameworku Next.js píší jeho tvůrci jako o nástroji, který není třeba konfigurovat a ve kterém jedním příkazem spustíte svoji reactí aplikaci. 

Důraz na jednoduchost kombinovanou s flexibilitou si pochvaluje i Robinův bývalý kolega z LMC, [Martin Krištof](https://www.linkedin.com/in/martin-kri%C5%A1tof-b6b78177/), se kterým jsme si o Next.js povídali. 

Co musíte umět pro začátek? Jaké všechny způsoby renderování tento inovativní framework umí? A nahradí Next.js ostatní backendy?
