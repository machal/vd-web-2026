# Podcast: S Martinem Držkou z Fakturoidu o utility třídách v CSS

S [Robinem](https://robinpokorny.com/) jsme [Martina](https://x.com/martindrzka) pozvali, protože nás zajímaly zkušenosti tvůrců oblíbeného fakturačního software s utility třídami.

Úplně nejlépe vás asi do problematiky uvede následující ukázka HTML kódu z nového rozhraní Fakuroidu:

```css
<div class="flex justify-between items-baseline">
  <h3 class="mb-0 h4 fw-n base truncate">
     Vystavné faktury
  </h3>
    <div class="swap-on-hover items-baseline ta-r">
        <span class="swap-on-hover__show base nowrap">
          1 210 Kč
          <small class="tiny"> vč. DPH</small>
        </span>
        <span class="swap-on-hover__hide ml-a h4 base nowrap">
          1 000 Kč
        </span>
    </div>
</div>
```

Proč zvolili tento přístup? Jaké to má výhody a nevýhody? Jak se dají utility třídy kombinovat s klasickými BEM komponentami? Poslechněte si nás:
