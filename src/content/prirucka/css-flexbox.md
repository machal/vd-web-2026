Flexbox je moderní způsob rozvržení webů, který umožňuje snadné zarovnání a distribuci prostoru mezi elementy.

## Základní koncepty

Flexbox pracuje s kontejnery a položkami:
- Flex container - rodičovský element
- Flex items - potomci flex containeru

## Hlavní vlastnosti

- `display: flex` - vytvoří flex container
- `flex-direction` - směr rozložení
- `justify-content` - zarovnání na hlavní ose
- `align-items` - zarovnání na vedlejší ose

## Praktické příklady

Flexbox je ideální pro:
- Centrování obsahu
- Rozložení navigace
- Responzivní layouty
