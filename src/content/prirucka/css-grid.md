CSS Grid je výkonný dvourozměrný layout systém, který umožňuje vytvářet komplexní rozvržení webů.

## Základní koncepty

Grid pracuje s mřížkou:
- Grid container - rodičovský element
- Grid items - potomci grid containeru
- Grid lines - čáry mřížky
- Grid tracks - řádky a sloupce

## Hlavní vlastnosti

- `display: grid` - vytvoří grid container
- `grid-template-columns` - definice sloupců
- `grid-template-rows` - definice řádků
- `grid-gap` - mezery mezi buňkami

## Praktické příklady

Grid je ideální pro:
- Komplexní layouty
- Kartové rozvržení
- Responzivní mřížky
