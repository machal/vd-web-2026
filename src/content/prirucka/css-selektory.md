CSS selektory umožňují vybrat konkrétní elementy v HTML dokumentu.

## Základní selektory

- Element selektor: `div`, `p`, `h1`
- Class selektor: `.class-name`
- ID selektor: `#id-name`
- Universal selektor: `*`

## Kombinované selektory

- Descendant: `div p`
- Child: `div > p`
- Adjacent sibling: `div + p`
- General sibling: `div ~ p`

## Pseudoselektory

- `:hover`, `:focus`, `:active`
- `:first-child`, `:last-child`
- `:nth-child()`

## Praktické příklady

Selektory jsou základním nástrojem pro styling webů.
