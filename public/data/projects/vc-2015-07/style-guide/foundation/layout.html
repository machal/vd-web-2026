Layout stránek 

<div class="vc-XXX">
  <div class="vc-container">
    <div class="vc-row">
      <div class="vc-content">
        <p>…</p>
      </div>
    </div>
  </div>
</div>

vc-XXX je nějaký konkrétní název např. head, main nebo foot.

V zásadě je to takto jednoduché, ale je lepší se podívat do layout.less nebo na konkrétní šablony.

Layout uvnitř stránek

Přebíráme princip 12sloupcového gridu podobný jako v Bootstrapu. Změna je v tom, že nepoužíváme třídy .col-X-Y pro vyznačení sloupců layoutu. Namísto toho máme vlastní sadu mixinů .vc-layout-col-base a .vc-layout-col(@columns).

Rozvžení ⅓ vs. ⅔ pro šířky okna větší než 600px získáme takto:

@media only screen and (min-width: 600px) {
  .vc-one-third {
    .vc-layout-col-base;
    .vc-layout-col(4);
  }

  .vc-two-thirds {
    .vc-layout-col-base;
    .vc-layout-col(4);
  }
}

Zlomy layoutu

Breakpointy má každá komponenta vlastní. Globální breakpointy jsou ve variable.less jen dva: 

// Bod, od ktereho se na obsah aplikuje layout
@vc-layout-start: 768px;

// Zlom, od ktereho se uz layout nezvetsuje
@vc-layout-full-start: 1200px;
