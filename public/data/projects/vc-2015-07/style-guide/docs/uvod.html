Tohle je příručka stylů pro nové weby Maternia, s. r. o.

Weby jsou postavené na frameworku Bootstrap a většina komponent z CSS základny (formuláře, tlačítka, tabulky…) je platná i zde. Pokud je použijete podle Bootstrap dokumentace a nebude něco fungovat, je možná potřeba je zapnout v bootstrap.less souboru.
