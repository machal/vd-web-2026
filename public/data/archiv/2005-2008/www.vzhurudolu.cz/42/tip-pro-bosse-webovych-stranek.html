Tip pro bosse webových stránek

20. červenec 2006, 11:23 Kategorie: Webdesign

Řídíte-li navštěvovaný web a tým, který se o něj stará, pak bych vám měl říkat manažer webových stránek a doporučit zajímavý blog a knihu.

Shane Diffily je jedním z vás a píše zajímavý blog a články k tématu:

Weby, které postrádají řízení se projevují řadou způsobů:

  • typicky se projevují velkou nesourodostí designu
  • předvádějí nedostatečné zaměření na obsah
  • využívají mizerných technologií

Shane je autorem knihy The Website Manager’s Handbook, která poslouží jak webdesignérům, kteří jsou nuceni někoho organizovat, tak manažerům, kteří jsou nuceni starat se o web.
