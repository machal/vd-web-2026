# Upozornění na dočasné soubory a skripty

**PRAVIDLO:** Při vytváření nebo úpravách kódu musím uživatele vždy upozornit na:

- Dočasné / ověřovací skripty (např. `check-*.js`, `verify-*.js`, testovací utility)
- Poznámky, TODO komentáře nebo debug výpisy zanechané v kódu
- Jakékoli soubory určené k pozdějšímu promazání

Před smazáním těchto věcí bez výslovného požadavku uživatele je vhodné se zeptat nebo upozornit: „Mám odstranit ověřovací skript X?“ / „Zůstávají v kódu debug poznámky – mám je vyčistit?“
